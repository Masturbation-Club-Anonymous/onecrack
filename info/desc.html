Описание продукта

Подписка включает в себя доступ к актуальной версии программного обеспечения для CS:GO. После покупки у вас будет возможность скачать программное обеспечение из личного кабинета на сайте. После скачивания и запуска приложения вам будет предложено пройти авторизацию - введите свои данные от аккаунта на этом сайте.


AIMBOT

  • Aimbot - помощь в наводке
  • Autofire - автоматическая стрельба
  • Silent - скрывает выстрелы
  • Doubletap - стреляет дважды за один тик
  • Remove visual punch - удаляет видимый разброс
  • Backtrack - стреляет в предыдущую позицию
  • Remove spread - режим NoSpread
  • Slide - движение ног
  • Fake duck - быстрые приседания
  • Knifebot - автоматическая атака ножа
  • Zeusbot - автоматическая атака зевса
  • Extended backtrack - расширение бектрека

ANTI-AIM

  • Standing / Moving / Air - срабатывание тех или иных условий ( в стоячем положении, в движении и в прыжке )
  • Pitch - направляет голову по оси Y
  • Yaw - направляет модель по оси Х
  • Yaw add - дополнительно разворачивает тело по оси Х
  • At fov target - разворачивает сторону фейка при противнике
  • Freestand - разворачивает модель на углах
  • Fake type - тип работы фейка ( фейк вокруг реала или наоборот )
  • Freestand fake - разворачивает фейк модель на углах
  • Fake lag - условия срабатывания фейклагов ( выкл. / всегда / при видимом противнике )
  • Base amount - базовое значение фейклага
  • Limit - лимит фейклага
  • Spin - разворачивает модель игрока вокруг своей оси
  • Jitter - разворачивает модель игрока в пределах, заданных пользователем
  • Anti-aim - включает функцию анти попадания
  • Fakelag always on - включает работу фейклагов независимо от каких-либо условий
  • Additional fakelag - дополнительный типсрабатывания фейклагов
  • Indicators - визуальные обозначения
  • Antiaim override - возможность переключить сторону АА кнопками
  • Back / Left / Right - настройки переключения по кнопке

WEAPONS

  • Auto / Scout / AWP / Heavy pistols / Pistols / Other - срабатывание тех или иных оружий
  • Hitchance - стреляет только с определнным шансом попадания
  • Min-damage - стреляет только если может нанести определенный урон
  • Head scale - берет в расчет определенный приоритет головы
  • Body scale - берет в расчет определенный приоритет тела
  • Hitboxes - срабатывание на определенные части тела
  • Multipoint - сканирование сразу нескольких определенных частей тела
  • Autostop - автоматическая остановка для повышения точности
  • Autoscope - автоматическое прицеливание

Config

  • Slot - выбрать настройки для загрузки
  • Load config - загрузить определенные настройки
  • Save config - сохранить определенные настройки
